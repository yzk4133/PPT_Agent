## 2. 数据问题

### 2.1 JSONL 格式错误

**症状**：

```
json.JSONDecodeError: Expecting property name enclosed in double quotes
```

**验证脚本**：

```python
import json

def validate_jsonl(file_path):
    """验证 JSONL 文件格式"""
    with open(file_path, 'r', encoding='utf-8') as f:
        for line_num, line in enumerate(f, 1):
            try:
                data = json.loads(line.strip())
            except json.JSONDecodeError as e:
                print(f"❌ Line {line_num}: {e}")
                print(f"   内容: {line[:100]}")
                return False

    print(f"✅ {file_path} 格式正确")
    return True

validate_jsonl('data/pretrain_hq.jsonl')
```

**常见错误**：

| 错误 | 正确 | 说明 |
|------|------|------|
| `'text'` | `"text"` | 必须用双引号 |
| `{text: "..."}` | `{"text": "..."}` | 缺少引号 |
| `{"text": ...}` | `{"text": "..."}` | 逗号后要有空格 |
| 中文逗号 | 英文逗号 | JSON 只支持英文标点 |

**症状**：


```python
    for line in f:
        data = json.loads(line)
```

**解决方案**：

1. **调整 max_length**：

2. **预处理数据**：
```python
def truncate_text(text, max_length=512, tokenizer):
    """智能截断"""
    tokens = tokenizer.encode(text)
    if len(tokens) <= max_length:
        return text

    # 在句子边界截断
    truncated = tokens[:max_length]
    # 找最后一个句号
    for i in range(max_length-1, 0, -1):
        if truncated[i] in tokenizer.convert_tokens_to_ids(['。', '！', '?']):
            return tokenizer.decode(truncated[:i+1])

    return tokenizer.decode(truncated)
```

3. **过滤过长样本**：
```python
def filter_long_samples(input_file, output_file, max_length=512):
    """过滤过长的样本"""
    tokenizer = AutoTokenizer.from_pretrained('../model/')

    with open(input_file, 'r') as f_in, open(output_file, 'w') as f_out:
        for line in f_in:
            data = json.loads(line)
            text = data.get('text', '')
            tokens = tokenizer.encode(text)

            if len(tokens) <= max_length:
                f_out.write(line)
```

## 3. 训练问题

### 3.1 Loss 不收敛

**症状**：

```
Loss 震荡、不下降或上升
```

**诊断**：

```python
def check_training_health(losses, window=100):
    """检查训练健康度"""
    if len(losses) < window:
        print("⚠️ 数据太少，无法判断")
        return

    recent_avg = sum(losses[-window:]) / window
    early_avg = sum(losses[:window]) / window

    print(f"早期平均 loss: {early_avg:.4f}")
    print(f"近期平均 loss: {recent_avg:.4f}")

    if recent_avg > early_avg * 1.1:
        print("❌ Loss 上升，可能学习率过大")
    elif recent_avg > early_avg * 0.95:
        print("⚠️ Loss 下降缓慢，可能学习率过小或数据质量差")
    else:
        print("✅ 训练正常")
```

**解决方案**：

1. **调整学习率**：
```bash
# Loss 震荡 → 降低学习率
--learning_rate 1e-7  # 从 5e-7 降到 1e-7

# Loss 下降慢 → 提高学习率
--learning_rate 1e-6  # 从 5e-7 提高到 1e-6
```

### 3.2 梯度爆炸/消失

```python
def check_gradients(model):
    """检查梯度状态"""
    total_norm = 0
    for name, param in model.named_parameters():
        if param.grad is not None:
            param_norm = param.grad.data.norm(2)
            total_norm += param_norm.item() ** 2

    total_norm = total_norm ** 0.5
    print(f"梯度范数: {total_norm:.4f}")

    if total_norm > 10:
        print("⚠️ 梯度爆炸")
    elif total_norm < 1e-5:
        print("⚠️ 梯度消失")
    else:
        print("✅ 梯度正常")
```

```python
    """调试推理"""
    model.eval()
    inputs = tokenizer(prompt, return_tensors='pt')
    inputs = {k: v.to('cuda') for k, v in inputs.items()}

    with torch.no_grad():
        outputs = model.generate(
            **inputs,
            max_new_tokens=50,
            do_sample=False,  # 关闭采样
            output_scores=True,
            return_dict_in_generate=True,

    # 解码
    decoded = tokenizer.decode(outputs[0], skip_special_tokens=True)
    print(f"输入: {prompt}")
    print(f"输出: {decoded}")

    # 检查 logits
    if hasattr(outputs, 'scores'):
        logits = outputs.scores[0]  # 第一个 token 的 logits

        top_probs, top_ids = probs.topk(5)
```

### 5.1 如何定位问题

```
1. 确认错误类型

   ├─ 数据问题 → 验证数据格式
   ├─ 训练问题 → 检查超参数

2. 获取详细信息
   ├─ 完整错误信息
   ├─ 数据样本
   └─ 配置参数

3. 最小化复现
   ├─ 减小数据量
   ├─ 简化模型
   └─ 单步调试

4. 尝试解决
   ├─ 查阅文档
```

### 5.3 单元测试编写

```python
import unittest
import torch

class TestMiniMind(unittest.TestCase):
    def test_model_forward(self):
        """测试模型前向传播"""
        from model.model_minimind import MiniMindForCausalLM, MiniMindConfig

        config = MiniMindConfig(hidden_size=512, num_hidden_layers=2)
        model = MiniMindForCausalLM(config)

        # 测试输入

        seq_len = 10
        input_ids = torch.randint(0, config.vocab_size, (batch_size, seq_len))

        # 前向传播
        outputs = model(input_ids)

        # 检查输出
        self.assertEqual(outputs.logits.shape, (batch_size, seq_len, config.vocab_size))

    def test_loss_computation(self):
        """测试损失计算"""
        from torch import nn

        # 模拟输出和目标
        logits = torch.randn(2, 10, 100)
        targets = torch.randint(0, 100, (2, 10))

        # 计算 loss
        loss_fct = nn.CrossEntropyLoss(reduction='none')
        loss = loss_fct(logits.view(-1, 100), targets.view(-1))
        loss = loss.view(2, 10).mean()

        # 检查
        self.assertFalse(torch.isnan(loss))
        self.assertFalse(torch.isinf(loss))

if __name__ == '__main__':
    unittest.main()
```

## 6. 总结

### 问题诊断决策树

```
训练出问题
    │
    ├─ 能启动吗？
    │   ├─ 否 → 环境问题 (检查依赖、CUDA)
    │   └─ 是 → 继续
    │
    ├─ 能加载数据吗？
    │   ├─ 否 → 数据问题 (验证格式、tokenizer)
    │   └─ 是 → 继续
    │
    ├─ Loss 正常吗？
    │   ├─ 否 → 训练问题 (学习率、梯度)
    │   └─ 是 → 继续
    │
    └─ 推理正常吗？
        ├─ 否 → 模型问题 (权重、生成参数)
        └─ 是 → 正常 ✅
```
