## 🐛 常见问题快速查找表

### 环境问题

| 问题 | 解决方案 | 文档位置 |
|------|---------|----------|
| CUDA 版本不匹配 | 检查 `torch.cuda.is_available()` | 08: 2.1 |
| 显存不足 (OOM) | 减小 batch_size，开启梯度累积 | 08: 2.3 |
| PyTorch 版本兼容 | 需 PyTorch ≥ 2.0 | 01: 4.1 |

### 数据问题

| 问题 | 解决方案 | 文档位置 |
|------|---------|----------|
| JSONL 解析错误 | 使用验证脚本检查格式 | 02: 4.1 |
| tokenizer 不匹配 | 确保 vocab_size 一致 | 02: 3.3 |
| 长度截断异常 | 调整 max_length | 02: 4.2 |
| 数据加载慢 | 增加 num_workers | 05: 7.4 |

### 训练问题

| 问题 | 解决方案 | 文档位置 |
|------|---------|----------|
| Loss 不收敛 | 检查学习率、数据质量 | 08: 3.2 |
| 梯度爆炸/消失 | 梯度裁剪、学习率调整 | 08: 3.3 |
| 训练速度慢 | Flash Attention、DDP | 08: 3.4 |
| DDP 同步问题 | 检查 DistributedSampler | 08: 3.5 |

### 模型问题

| 问题 | 解决方案 | 文档位置 |
|------|---------|----------|
| 推理结果异常 | 检查权重加载、tokenizer | 08: 4.1 |
| 幻觉问题 | DPO 训练、温度调整 | 08: 4.2 |
| 性能下降 | 检查评估指标、数据分布 | 08: 4.3 |

## 💡 学习建议

### 初学者

1. **按顺序阅读** P0 文档
2. **动手实践**：运行代码，修改参数
3. **记录笔记**：写下自己的理解
4. **提问讨论**：与他人交流

### 进阶学习

1. **深入原理**：阅读 P1 文档
2. **对比研究**：与其他项目对比
3. **改进优化**：尝试改进模型
4. **撰写文档**：输出倒逼输入

### 面试冲刺

1. **快速复习**：使用 09 的复习清单
2. **模拟面试**：自己回答问题
3. **查漏补缺**：针对性学习
4. **保持自信**：理解 > 记忆
