## 🚀 三步快速训练

### 步骤 1: 下载预训练数据

```bash
# 创建数据集目录
mkdir -p dataset

# 下载核心数据集（从 ModelScope）
# 方式一：使用 git clone
git clone https://www.modelscope.cn/datasets/gongjy/minimind_dataset.git dataset/

# 方式二：手动下载
# 访问 https://www.modelscope.cn/datasets/gongjy/minimind_dataset/files
# 下载这两个文件即可：
# - pretrain_hq.jsonl (1.6GB)
# - sft_mini_512.jsonl (1.2GB)
```

### 步骤 2: 开始训练

```bash
# 进入训练目录
cd trainer

# 2.1 预训练（约1.1小时）
python train_pretrain.py \
  --data_path ../dataset/pretrain_hq.jsonl \
  --epochs 1 \
  --batch_size 32 \
  --max_seq_len 512

# 输出文件: ../out/pretrain_512.pth

# 2.2 监督微调（约1小时）
python train_full_sft.py \
  --data_path ../dataset/sft_mini_512.jsonl \
  --epochs 1 \
  --batch_size 32 \
  --max_seq_len 512

# 输出文件: ../out/full_sft_512.pth
```

### 步骤 3: 测试模型

```bash
# 回到项目根目录
cd ..

# 运行评估脚本
python eval_model.py \
  --model_mode 1 \
  --hidden_size 512 \
  --num_hidden_layers 8

# model_mode 说明:
# 0 = 预训练模型（只会词语接龙）
# 1 = SFT对话模型（可以聊天）
# 2 = RLHF模型（更礼貌）
# 3 = Reasoning模型（有推理能力）
```

## 🔧 自定义训练

### 修改训练参数

编辑 `trainer/train_pretrain.py`：

```python
# 常用参数
parser.add_argument("--batch_size", type=int, default=32)
parser.add_argument("--learning_rate", type=float, default=5e-4)
parser.add_argument("--epochs", type=int, default=1)
parser.add_argument("--max_seq_len", default=512, type=int)
```

### 多卡加速（如果你有多张GPU）

```bash
# 使用 2 张卡
torchrun --nproc_per_node 2 train_pretrain.py

# 使用 4 张卡
torchrun --nproc_per_node 4 train_pretrain.py
```

## ❓ 常见问题

### Q1: 训练需要多久？
A: 使用 MiniMind2-Small 配置，单卡3090约2小时。详见上方表格。

### Q2: 可以用 CPU 训练吗？
A: 可以，但非常慢。预计需要 10-20 小时。强烈建议使用 GPU。

### Q3: 如何查看训练进度？
A: 终端会实时打印 loss 和进度。添加 `--use_wandb` 参数可使用 WandB 可视化。

### Q4: 训练中断了怎么办？
A: 每100步自动保存检查点。重新运行训练会从最近的检查点继续。

### Q5: 显存不足怎么办？
A: 减小 `batch_size` 或 `max_seq_len`：
```bash
python train_pretrain.py --batch_size 16 --max_seq_len 256
```
