## 📊 快速参考

### 训练时间速查

| 模型 | 预训练 | SFT | LoRA | 总时间 | 总成本 |
|-----|--------|-----|------|--------|--------|
| MiniMind2-Small (26M) | 1.1h | 1h | 15min | 2.25h | ¥3 |
| MiniMind2 (104M) | 3.9h | 3.3h | 30min | 7.5h | ¥10 |

### 关键参数速查

```python
# 最小模型（快速）
hidden_size = 512
num_hidden_layers = 8
参数量 = 26M

# 标准模型（推荐）
hidden_size = 768
num_hidden_layers = 16
参数量 = 104M

# MoE模型（高效）
hidden_size = 640
num_hidden_layers = 8
use_moe = True
参数量 = 145M
```

### 常见命令速查

```bash
# 预训练
python train_pretrain.py --data_path ../dataset/pretrain_hq.jsonl --epochs 1

# SFT训练
python train_full_sft.py --data_path ../dataset/sft_mini_512.jsonl --epochs 1

# LoRA训练
python train_lora.py --data_path ../dataset/lora_custom.jsonl

# 测试模型
python eval_model.py --model_mode 1 --hidden_size 512

# 多卡训练
torchrun --nproc_per_node 2 train_pretrain.py
```
