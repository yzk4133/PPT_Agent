# 研究工作指南

## Level 1: 快速指南

### 核心步骤（3 步）

1. **理解需求** → 识别需要研究的关键信息
2. **搜索收集** → 使用工具获取相关资料
3. **整理归纳** → 提炼核心要点和数据

### 输出格式

```
研究资料：

**背景知识**：
- 要点1
- 要点2

**关键数据**：
- 数据1（来源说明）
- 数据2（来源说明）

**相关案例**：
- 案例1简述
- 案例2简述
```

---

## Level 2: 详细指南

### 步骤 1: 理解需求

**信息类型识别**：
- **概念解释**: 需要定义、原理、发展历程
- **技术细节**: 需要技术参数、实现方式
- **数据支撑**: 需要统计数据、实验结果
- **案例说明**: 需要实际应用、成功案例

### 步骤 2: 搜索策略

**搜索关键词构建**：
```
主题 + 核心概念 + 具体方向
```

示例：
- "深度学习 图像识别 原理"
- "CNN 架构 应用案例"
- "AI 准确率 对比数据"

**工具选择**：
- `web_search`: 通用网络搜索
- `fetch_url`: 获取特定URL内容
- `weixin_search`: 微信公众号文章（中文资料）

### 步骤 3: 信息整理

**信息质量检查**：
- ✅ 来源可信度（官方、学术、知名媒体）
- ✅ 数据时效性（优先近2年数据）
- ✅ 内容相关性（与主题直接相关）
- ✅ 信息完整性（有数据、有案例）

---

## Level 3: 高级技巧

### 技巧 1: 并行搜索

同时搜索多个角度，交叉验证信息：

```python
search_queries = [
    f"{topic} 原理",
    f"{topic} 应用案例",
    f"{topic} 最新进展",
    f"{topic} 数据统计"
]
```

### 技巧 2: 深度挖掘

从初步结果中发现新的研究方向：

1. 从结果中识别关键术语
2. 对关键术语进行二次搜索
3. 整合多来源信息

### 技巧 3: 数据验证

- 多源对比：同一数据至少2个来源
- 标注说明：明确示例数据vs真实数据
- 时间标注：注明数据时间点
