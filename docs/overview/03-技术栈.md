# 技术栈

## 🤖 智能体框架

| 技术 | 版本 | 用途 | 说明 |
|------|------|------|------|
| **Google ADK** | 1.5.0+ | 智能体开发 | 多 Agent 框架，支持 A2A 和 MCP |
| **A2A Protocol** | - | Agent 通信 | Agent-to-Agent 通信协议 |
| **MCP** | - | 工具调用 | Model Context Protocol |
