## [0.1.0] - 2026-02-02

### 新增
- 🎉 首次公开发布
- ✨ 完整的多Agent协作架构
- 🤖 内容生成Agent（支持大纲和内容生成）
- 🎨 设计Agent（配色、字体、布局建议）
- 💻 代码生成Agent（生成python-pptx代码）
- 📋 任务协调器（任务分解和调度）
- 🔧 配置管理系统（支持YAML配置）
- 💾 上下文记忆系统（Agent对话历史）
- 📄 PPT模板系统（内置多种模板）
- 🌐 FastAPI Web服务（RESTful API）
- 📊 性能监控模块
- 🧪 完整的测试框架
- 📚 详细的项目文档

### 技术栈
- 后端：Python 3.11 + FastAPI
- AI：Claude API + 自研MiniMind
- PPT：python-pptx
- 部署：Docker + Docker Compose
